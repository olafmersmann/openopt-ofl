# OFL - Optimisation feature library
Collection of features to characterise (black-box) optimisation problems, based on some sampling method.

## Usage

To rebuild `docs/features.html` and `docs/index.html` run

```sh
uv run yaml_to_html.py
```

If all goes well, the script outputs nothing and updates the two files in place.
